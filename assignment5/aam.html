Advanced Array Methods

Arrays are everywhere, from simple matrices to complex list of objects holding every little detail about each product on a page. We have to deal with them day to day and often when we operate on them, we write out the same lines of code instinctively without giving much attention about the possible forms they can take. The way you write them can determine how easy it is for others to read and understand your implementation.

Luckily in JavaScript we have some nice built in functions for dealing with arrays that can simplify your code, make it easier to read and in the end, make it easier to debug in case something goes off the rails. These functions are "Advanced Array Methods".


Now to demonstrate each and every function we have, we are going to need some data to work with.

There are two arrays and I am going to do some array operations to them.

Example

const companies=[
{name:"one", category: "finance", start: 1981, end: 2003},
{name:"two", category: "retail", start: 1982, end: 2004},
{name:"three", category: "auto", start: 1983, end: 2005}
];
const ages=[33,12,20,16,5,54,21,44,61,13,15,45,84,23];

1.Filter

const canDrink = ages.filter(age=> age >= 21); console.log(canDrink);

2.Push

ages.push(45) console.log(ages);

3.Pop

ages.pop(); console.log(ages)

4.Find

let two=companies.find(company=>company.name=="two"); console.log(two);

5.Sort

let sortAges=ages.sort(); console.log(sortAges);

6.Map

const newAges=ages.map(function(age){ return age+1; }); console.log(newAges);

7.Shift

var newAges2= ages.shift(); console.log(newAges2); console.log(ages);

8.Unshift

ages.unshift(12); console.log(ages);
