1.Window Object

When ‘this’ is used in a web browser, the window object is referenced.

window is defined in a web

console.log(this==window)//true

window is undefined in Node

console.log(this==window)//false


2.Global Object

When ‘this’ is used outside function it is referring to a global object.

Example

var name = 'John';//global object
function foo() {
console.log(this.name);
}
foo()

Because name is a global object, you can access it in the function foo()
