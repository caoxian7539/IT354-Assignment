Here is an example about how I can access and set values of objects

Example

const myCar = {
name: "Audi",
model : "A4",
color : "Black",
get showDetails() {
console.log("my name is "+ this.name+ " and my model is "+ this.model);
}
}
myCar.showDetails//my name is Audi and my model is A4

In an object, you can also set function in it. In the example, I set a showDetails() function, and I can use it to access the values in the object


What if I want to change a car?

Example

myCar.name="Honda";
myCar.showDetails;//my name is Honda and my model is A4

I can reset the name of myCar to Honda and in showDetails() the output will chage correspondingly
