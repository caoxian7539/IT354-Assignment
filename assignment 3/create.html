There are many ways to create new object.

1.Creating objects with 'new Object()'

Example

const myCar=new Object();
myCar.name= "Audi";
myCar.mode="A4";
myCar.color="Black";

I created a new object myCar with three name:value pairs.

Remember that 'O' of 'Object' has to be in uppercase.


2.Creating objects with object literals

Example

const myCar={name: "Audi", model: "A4", color: "Black"};

In my opinion, this is the most direct and intuitive way to create an object, and It saves lines and codes.


3.Creating objects with constructors

Example

function myCar(name,model,color){
this.name=name;
this.model=model;
this.color=color;
}
const oneCar=new myCar("Audi", "A4", "Black");

I created a constructor myCar with three arguments of it. After that, I created a new object: oneCar of myCar that has its own values corresponding to three arguments


4.Creating objects with classes (special functions):

Example

class myCar{
constructor(name,model,color){
this.name=name;
this.model=model;
this.color=color;
}}
const oneCar=new myCar("Audi", "A4", "Black");

This is much the same as the last one. Notice the key words 'class' and 'consturctor'.

Also pay attention to the brace '{}' and bracket '()'.
