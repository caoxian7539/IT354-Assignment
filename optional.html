Optional Arguments: It’s a functions in which certain arguments are optional. You can set a default value for any argument,so if the argument is not present, the default value will be used.

function add(a=100,b){
Return a+b; }
add(2,3);//5

JavaScript ignores all extra arguments, and if we don't pass enough then the missing ones are undefined.

function square(a){
Return a*a;
}square(4,’sad’,’but’)//16,ignores all extra arguments
