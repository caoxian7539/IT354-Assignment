			1.Numbers are written with or without decimals:10.5 100, don’t have to initiate them with int, double or float.
			var a=6, var b=100.102
			

			Scientific Notation, var a=1e6 -> 1000000 or 1*10^6
			var b=1.4e6 -> 1400000 or 1.4*10^6

			2.Unique Numbers
			Infinity or -Infinity
			Infinity is useful for knowing when you've gone higher or lower than the largest possible number in JS
	        Numbers out of (-1.7976931348623157E+10308,1.7976931348623157E+10308) will be defined Infinity or -Infinity
			

			3.Strings
			JavaScript strings are used for storing and manipulating text.
			var x = "Kobe Bryant", var y='Lebron James', var z=x+y
			

			The sequence \"  inserts a double quote in a string:
			var x = "We are the so-called \"Vikings\" from the north."// We are the so-called "Vikings" from the north.
			


			4.Unary Operators: typeof
			typeof
			This is super helpful when debugging or working with data that changes
			typeof 2.5 // returns number
			Unary Operators: Unary Plus, Minus
			+, -
			Plus: converts strings of numbers, booleans, and null to numbers
			Minus: Does the same thing as plus, but negates
			Unary Operators: Logical Not:!•	
			Returns the boolean equivalent with negation
			
			Unary Operators: delete
			example: delete myArray[0] // returns true
			Returns true if it deleted a property or if property doesnt exist
			Doesn't work with variables and functions...
			var a=[1,2,3,4,5]
			delete a[0]
			

			5.Hoisting
			It is when a browser/engine compiles variables and named-functions immediately
			Hoisting is JavaScript's default behavior of moving all declarations to the top of the current scope (to the top of the current script or the current function). In JavaScript, a variable can be declared after it has been used.
			In other words; a variable can be used before it has been declared.
	
			carName = "Volvo"
			console.log(carName)
